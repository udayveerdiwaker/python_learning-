# Module 01: Python Basics 🐍

In this module, you will learn the foundational concepts of Python. Before we can build APIs or work with databases, we must understand how Python stores data, processes logic, repeats tasks, and organizes code.

---

## 📚 Topics Covered

1. **Variables and Data Types**: Storing numbers, text (strings), booleans (`True`/`False`), and floating-point decimals.
2. **Lists**: Storing ordered sequences of values (like a shopping list).
3. **Dictionaries**: Storing key-value pairs (like a telephone directory where a name maps to a number).
4. **Conditional Statements (`if`/`elif`/`else`)**: Making decisions in code based on conditions.
5. **Loops (`for` and `while`)**: Repeating tasks automatically.
6. **Functions**: Creating reusable blocks of code to perform specific tasks.
7. **Basic Object-Oriented Programming (OOP)**: Creating classes and objects to model real-world concepts.

---

## 🗂️ Files in this Module

- `basics.py`: A fully commented script demonstrating every topic in detail. Run it using:
  ```bash
  python basics.py
  ```
- `exercises.py`: Practice tasks for you to complete. We've included automated tests at the bottom of the file to verify your solutions! Run it using:
  ```bash
  python exercises.py
  ```
- `challenge.py`: A final challenge to test your knowledge of variables, loops, lists, and functions. Run it using:
  ```bash
  python challenge.py
  ```

---

## ✍️ Practice Exercises Overview

Inside `exercises.py`, you will implement:
1. `calculate_area(length, width)`: Returns the area of a rectangle.
2. `get_grade(score)`: Returns a grade letter (A, B, C, D, F) based on a numeric score.
3. `sum_even_numbers(numbers_list)`: Filters and sums all even numbers in a list.
4. `add_to_phonebook(phonebook, name, number)`: Inserts a name and phone number into a dictionary and returns it.

## 🏆 Challenge Task: Text-Based RPG Inventory & Battle simulator
Inside `challenge.py`, you will build a mini console-based Role Playing Game (RPG) inventory manager and battle logger. The script will let you:
1. View player status and inventory list.
2. Add items to the inventory while respecting a maximum capacity.
3. Simulate battles where health changes, items are consumed, and logs are kept in lists.
